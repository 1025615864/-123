## 2. 本次交付范围（可验收）

### 2.1 核心交付能力（概览）

系统已具备：

- AI 法律咨询（同步 + SSE 流式）
- 文书生成
- 新闻资讯（含 News AI 标注）
- 论坛社区（含审核/通知联动/回收站/草稿箱）
- 律所/律师服务（认证、预约、支付、接单、留言、完成、评价）
- 支付与订单（余额支付闭环；支付宝支付链接生成；回调验签与审计留痕）
- 律师结算与提现（律师端 + 管理端，含 CSV 导出、脱敏）
- 商业化与配额（VIP / 次数包 / 每日额度，系统设置可配置）
- 管理后台（运营/风控/系统设置/日志等）

### 2.2 律师结算/提现（M1）

- 律师钱包/收入明细（冻结期/结算）
- 提现申请、提现记录
- 收款账户（银行卡）管理（字段加密存储 + 脱敏展示）
- 管理端提现审核/驳回/打款完成或失败标记
- 管理端统计与 CSV 导出（收入/提现）

### 2.3 商业化与定价（M2）

- VIP 会员：可购买，权益由系统设置可配置
- 次数包：AI 咨询/文书生成次数包可购买
- 配额策略：每日额度 + 次数包余额 + VIP 权益的执行与扣减
- 管理端可配置价格与权益，用户侧展示与下单金额一致

### 2.4 用户角色与权限模型（摘要）

- **游客**
  - 可浏览公开内容（新闻/论坛列表等）
  - AI 咨询存在游客配额/限流（按 IP 维度）
- **普通用户（user）**
  - 登录后可发帖/评论/收藏/订阅等
  - 可使用个人中心、通知、法律工具等
- **律师用户（lawyer）**
  - 通过“律师认证申请 → 管理员审核”后获得律师身份
  - 可访问律师工作台处理用户咨询预约（接单/拒单/完成、留言等）
  - 可访问律师结算与提现能力：钱包、收入明细、收款账户、提现申请与记录
- **管理员（admin / super_admin）**
  - 可访问管理后台：新闻管理、论坛管理、用户管理、系统设置、日志、通知、知识库等

- 补充协议模板（范围/里程碑/费用口径）：`docs/_archive/补充协议模板.md`

### 3.2 建议验收顺序（摘要）

1. 基础：登录/注册/合规页面、管理后台、Swagger
2. 律师服务闭环：预约 → 支付 → 律师接单 → 留言 → 完成 → 评价
3. 律师结算/提现：收入记录 → 冻结期 → 解冻结算 → 提现申请 → 管理端审核/打款 → CSV 导出
4. 商业化与配额：
   - 管理端配置价格/权益
   - 用户侧价格展示与下单金额一致
   - 购买 VIP/次数包后权益/余额生效
   - `/chat` 与 `/documents` 的配额扣减与次数包兜底
